Burrows-Wheeler Transform
The Burrows-Wheeler Transform is a tool used in compression that aims to group together characters that are the same. This program will compress a text file. It will show the difference between using the Burrows-Wheeler Transform with the Lyndon Factorisation and without. It will allso show how individual compression tools effect how effective the compression can be.
